# Notas del curso de JS

### Tipos de datos en JS


### Mutabilidad
    La mutabilidad es cuando le re asignas a una variable u objeto un nuevo valor o el estado.

### Objetos en JS
    Un objeto es una agrupacion de datos y estos datos haran sentido entre si
        Ejemplo:
                const obejeto = {
                    propiedad: valor de la propiedad,
                }
    los objetos pueden tambien almacenar funciones.
    Dato: uno tambien puede llamar alguna de sus propiedades usando el .

### Arreglos
    Un arreglo es una coleccion de elementos. Los arreglos guardan data como elementos y los 
    regresan cuando lo necesites.
        Ejemplo:
                const arreglo = [una variable,string,numero,objeto];
    para agregar a un arreglo vacio se hace el llamdo del arreglo y se le agrega el 
    .push()
        Ejemplo: arreglo.push(10);
